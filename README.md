
# Binary Search Tree in Java

This is a Java implementation of a Binary Search Tree (BST) supporting:

- Insert a node
- Search for a node
- Delete a node
- Inorder Traversal

## 📁 File Structure

- `BinarySearchTree.java` — Core logic and main method to test

## 🚀 How to Run

```bash
javac BinarySearchTree.java
java BinarySearchTree
